# Dev Hub

A high-performance, minimalist dashboard designed for developers to manage and access technical resources across various domains including AI, Machine Learning, Cybersecurity, and IoT.

## Live Demo
Access the application here: [https://sahil002620q.github.io/dev-hub/](https://sahil002620q.github.io/dev-hub/)

## Overview
Dev Hub serves as a centralized command center for developers. It eliminates the need for maintaining extensive browser bookmarks by providing a categorized, searchable, and customizable interface for frequently used tools and documentation.

## Core Features
* **Categorized Resource Management:** Tools are organized into specialized domains such as Code, AI/LLM, ML & Data, Security, and IoT.
* **Glassmorphism UI:** A modern interface utilizing high-end CSS blur effects and radial gradients.
* **Personalized Experience:**
    * **Dark Mode:** A deep black theme optimized for low-light environments.
    * **Accent Customization:** Switch between multiple accent colors including Blue, Green, Amber, Pink, and Violet.
    * **Density Control:** Support for Compact, Comfortable, and Spacious grid layouts.
* **Advanced Filtering:** Multi-select domain filtering allows for the isolation of specific tech stacks.
* **Local Persistence:** All user preferences (theme, accent, favorites, and density) are saved locally via the Browser Storage API.
* **Instant Search:** Real-time search functionality across titles, descriptions, and categories.

## Technical Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom properties (CSS variables), Flexbox, CSS Grid, and sophisticated animations.
* **JavaScript (Vanilla):** State management, DOM manipulation, and local storage integration.
* **Google S2 API:** Dynamic favicon retrieval for real-time icon rendering.
